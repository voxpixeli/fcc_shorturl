URL Shortener Microservice

usage

to shorten an address, use ttps://knotty-headlight.glitch.me/shorten/, followed by the address
example input: https://knotty-headlight.glitch.me/shorten/https://en.wikipedia.org/wiki/FreeCodeCamp
result: {"originalUrl":"https://en.wikipedia.org/wiki/FreeCodeCamp","shortUrl":"H7f"}

using a shortened address
example input: https://knotty-headlight.glitch.me/H7f
redirects to: https://en.wikipedia.org/wiki/FreeCodeCamp
